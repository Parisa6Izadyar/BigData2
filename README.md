# Big Data River Project

This repository contains an academic Big Data project focused on forecasting water levels in the Arno River using historical hydrological and environmental data.

## Project Overview
The project investigates water availability prediction using time series analysis on river-related data. The study focuses on the Arno River in Italy and explores how hydrometry, average temperature, and precipitation affect water levels over time.

## Dataset
The project is based on more than 20 years of observations related to:
- River water levels (hydrometry)
- Average temperature
- Average precipitation

## Methods
- Time series analysis
- SARIMAX modeling
- Forecasting and trend evaluation

## Repository Structure
- `notebooks/` → Jupyter notebook for the project
- `results/` → plots, forecasts, and result images
- `data/` → dataset files if included
- `README.md` → project documentation
